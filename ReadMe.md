# 1. 功能
1. 双击选词翻译需求
2. 双击选词时间戳转格式化日期
# 2. 如何打包
1. gradle先clean，再build
2. 在项目目录下路径，genCloud\build\idea-sandbox\plugins，可以找到同名项目文件夹
3. 右键打包成zip文件
4. 导入idea，通过硬盘安装插件
